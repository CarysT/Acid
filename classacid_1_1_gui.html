◆ SetNinePatches()

void acid::Gui::SetNinePatches ( const Vector4f &  ninePatches)
inline

Sets nine the patches value size, if zero it will be disabled. This value represents the top-left and bottom-right content bounds. 9-patch/9-slicing allows for a single section of a image to be scale with corners and edges kept in the screens aspect ratio.

Parameters
ninePatchesThe values, x/y being to top left corner and z/w bottom right for the scalable section.
